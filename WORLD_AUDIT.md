# World Identity & Mechanics Audit

## Summary
Each world now has a clear identity and progression path. The first 10 minutes of gameplay (Worlds 1-2) should successfully hook players with:
1. Clear skill progression (basics → advanced technique)
2. Rewards at every stage
3. Visual identity unique to each world
4. Satisfying boss encounters with unique mechanics

---

## World 1: THE AEGIS CORE (Shields & Precision)
**Theme**: Learning the fundamentals, breaking through barriers

### Stages:
- **1-1**: The Basics (4 hits) - Tutorial: single zone, learn timing
- **1-2**: Reversal (5 hits) - Zone reverses after each hit, master direction changes
- **1-3**: Drift (6 hits) - Zone moves around ring, track target
- **1-4**: Precision (7 hits, 2 targets) - Two zones that shrink, demand precision
- **1-5**: Frenzy (6 hits, 4 targets) - Fast-paced with multiple targets, emotional high
- **1-6 BOSS**: The Aegis Core (99 hits) - Three shield layers that break away, exposing the core

### Identity:
✅ Strong progression curve
✅ Tutorial loops into increasing complexity
✅ Players feel competent by stage 1-5
⚠️ **TODO**: Aegis boss should have visible core element similar to Phoenix boss
  - Add glowing shield orb in center
  - Shields pulse and break away sequentially
  - Core becomes targetable after shields break

### Score Values (post-update):
- OK = 2 pts
- Good = 5 pts
- Perfect = 10 pts
- Filthy Perfect = 15 pts + multiplier boost

### Notes:
- World 1 has FREE RESTART on failure (tutorial mercy)
- Excellent pacing for first 10 minutes
- Boss intro is handled by cinematic sequence

## World 2: THE PRISM (Patterns & Sequences)
**Theme**: Reading patterns, precision reading, mirror mechanics

### Stages:
- **2-1**: Refract (5 hits) - Corner point mechanics, learn precision hit zones
- **2-2**: Twin Signal (5 hits) - Two mirrored zones, must hit in sequence
- **2-3**: Fracture (3 hits) - Hit once to split, then finish fragments
- **2-4**: Cascade (3 hits) - Fragments move (drift), higher precision
- **2-5**: Critical Angle (6 hits, shrinking, reversing) - Combines all mechanics
- **2-6 BOSS**: The Prism (99 hits) - Rotation and sequence phases, arena rotates

### Identity:
✅ Pattern recognition core mechanic
✅ Introduction of split/fragment targets
✅ Boss arena is unique (rotating ring)
⚠️ **TODO**: Prism core element
  - Add faceted crystal core rotating in center
  - Facets light up as player hits correct sequence
  - Core rotates and changes color as phases progress

### Difficulty Spike:
- Good step up from World 1
- Split targets require new mental model
- Boss is mechanically distinct (arena rotation)

### Notes:
- First truly challenging world
- Normal failure mechanics apply (lose lives)
- Should take 10-15 minutes average to complete

## World 3: THE SPECTRE (Echo Mechanics & Prediction)
**Theme**: Delayed reaction, reading echoes, timing with ghosts

### Stages:
- **3-1**: Echo Arrival (2 hits) - Cyan ghost trails behind, echo must hit first
- **3-2**: Echo Field (3 hits) - Only delayed echo can score, track cyan
- **3-3**: Split Field (4 hits) - Orange = real orb, cyan = echo, mixed targets
- **3-4**: Echo Drift (4 hits) - Cyan zones drift while delaying
- **3-5**: Cross Signal (5 hits) - All mechanics combined
- **3-6 BOSS**: The Spectre (8 hits) - Phases through echo patterns

### Identity:
✅ Unique ghost/echo mechanic differentiates from other worlds
✅ Requires predictive thinking
✅ Visual distinction with cyan ghost targets
⚠️ **TODO**: Spectre core element
  - Add translucent ghost core pulsing in center
  - Core flickers between visible/invisible
  - Splits into echo when boss attacks

### Difficulty:
- Significant mindset shift from Worlds 1-2
- Players must predict where echo will be
- More forgiving (fewer total hits needed)

### Notes:
- 3-1 and 3-2 are excellent teaching stages
- Echo mechanic is immediately engaging
- Boss should be visually distinct from Aegis/Prism

## World 4: THE CORRUPTOR (Phantom Zones & Detection)
**Theme**: Identifying real from corrupted, signal integrity, survival

### Stages:
- **4-1**: Corrupt Signal (6 hits) - One real zone, one phantom (fake)
- **4-2**: Static Burst (7 hits, 2 zones, reversing) - Multiple zones with phantoms
- **4-3**: Fragment Storm (5 hits, split) - Hit once to fragment, one shard is ghost
- **4-4**: Phase Shift (8 hits, 2 zones, shrinking, reversing) - Maximum complexity pre-boss
- **4-5**: System Overload (10 hits, 3 zones) - All real, pure execution test
- **4-6 BOSS**: The Corruptor (99 hits) - Nodes destroy, exposing corrupt core

### Identity:
✅ "Read the signal" core theme
✅ Visual phantom effect (desaturated/glitchy)
✅ High-risk, high-reward mechanic
⚠️ **TODO**: Corruptor core element
  - Add pulsing corruption orb in center
  - Shows visual "corruption" spreading/receeding
  - Nodes rotate around core, each must be destroyed
  - Core shifts between real and phantom

### Difficulty:
- World 4 is the skill wall
- Phantom mechanic requires careful reading
- 4-5 is pure execution (no tricks)

### Notes:
- Players who beat World 4 have mastered core mechanics
- Corruptor should feel menacing/glitchy
- Boss nodes should have clear hit feedback

- **5-5**: Null Storm (10 hits, 3 zones, shrinking, blackout) - Maximum pressure
- **5-6 BOSS**: The Null Gate (99 hits) - Blind phases, trust required

### Identity:
✅ Blackout mechanic is visually striking
✅ Tests pure muscle memory and rhythm
✅ Pentagon shape is unique
⚠️ **TODO**: Null Gate core element

## Cross-World Observations

### Target Consistency (Unless World-Unique):
✅ Orange targets = real zone (hiteable for score)
✅ Cyan targets = echo/ghost (world 3-5 special)
✅ Red/Corrupt = phantom (world 4 special)
✅ All targets scale/shrink consistently
✅ All target motion is smooth and predictable

### Boss Cores - Action Items:
Each boss should have a unique visual core element in the center of the screen:
1. **Aegis** - Shield orb (breaks away in phases)
2. **Prism** - Faceted crystal (rotates, changes color)
3. **Spectre** - Ghost core (flickers, splits)
4. **Corruptor** - Corruption sphere (spreading/receeding)
5. **Null Gate** - Void shadow (invisible phases, distortion)
6. **Solar Core** - Inferno globe (heat distortion, color progression)

Each core should:
- Visually indicate boss health/phase
- Provide feedback on player damage
- Create memorable aesthetic
- Stay within ~80-100px diameter (center of screen)

### First 10 Minutes (Tutorial + World 1):
✅ **Perfect structure**: 1-1 through 1-5 teaches all core skills
✅ **Free restart on failure** encourages experimentation
✅ **Satisfying progression** from tutorial to boss
✅ **Clear feedback** on hits (audio + visual)
✅ **New scoring system** makes progression feel rewarding

### Worlds 2-6:
✅ Each introduces ONE core new mechanic
✅ Difficulty ramps smoothly
✅ Boss fights are distinct from regular stages
⚠️ Boss cores need visual overhaul (all should have unique center orb like Phoenix)
